## Build Setup

```bash
# 安装依赖
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
-- 服务器通过pm2 启动 `pm2 start ./node_modules/nuxt/bin/nuxt -- start`
# 生成静态目录
$ npm run generate
```
